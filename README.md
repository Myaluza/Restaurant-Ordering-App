# Restaurant Ordering App

## Description
A web application that allows users to browse restaurant menus, add items to their cart, and place food orders online.

## Features
- View menu items with descriptions and prices
- Add/remove items from shopping cart
- Calculate order total
- Place orders with customer information
- Responsive design for mobile and desktop

## Technologies
- HTML5
- CSS3
- JavaScript

## Usage
1. Select items from the menu
2. Review your cart
3. Enter payment details
4. Complete your order
